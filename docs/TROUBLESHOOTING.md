# Troubleshooting Guide

Common issues and solutions for Deep-Live-Cam Chrome Extension.

## Quick Fixes

### Extension Not Loading
1. Go to `chrome://extensions/`
2. Find Deep-Live-Cam
3. Click "Reload" button
4. If still not working, click "Remove" and reinstall

### Camera Not Working
1. Check camera permissions in browser
2. Allow camera access for the extension
3. Try a different camera in system settings
4. Restart Chrome

### Face Detection Issues
- Ensure good lighting
- Face should be clearly visible
- Avoid obstructions on face

## Error Messages

### "Models not loaded"
Download AI models in extension popup

### "Camera access denied"
1. Click camera icon in address bar
2. Allow camera access
3. Refresh the page

### "Extension context invalidated"
Reload the extension and try again

## Performance Issues

### Slow on Low-End Devices
1. Lower resolution (480p)
2. Reduce FPS to 10
3. Disable face enhancer
4. Use LITE version

### Memory Issues
1. Close other tabs
2. Reduce quality settings
3. Use LITE version
