# Quick Start Guide

Get up and running with Deep-Live-Cam Chrome in 5 minutes!

## Step 1: Download

Choose your version:

### LITE (Recommended for most)
- No downloads needed
- Works immediately
- ~40KB size

### MINI (Best quality)
- Requires ~250MB model download
- Higher quality face swapping
- Best for high-end devices

## Step 2: Install

1. Download the ZIP from [releases](https://github.com/yjdudeplz-web/deep-live-cam-chrome/releases)
2. Extract the ZIP file
3. Open Chrome and go to `chrome://extensions/`
4. Enable **Developer mode** (top right toggle)
5. Click **Load unpacked**
6. Select the extracted folder

## Step 3: Use

### Basic Face Swap

1. Click the extension icon in Chrome toolbar
2. Click "Open Face Swap"
3. Select a face image (clear, front-facing works best)
4. Choose your camera
5. Click "LIVE" to start

### Video Chat

1. Click extension icon
2. Select a video chat site (Omegle, OmeTV, etc.)
3. Allow camera access if prompted
4. Select your face image
5. Start chatting!

## Quick Settings

| Setting | Default | Best For |
|---------|---------|----------|
| Quality | Balanced | Most users |
| Resolution | 720p | Good quality |
| FPS | 15 | Smooth video |

## Troubleshooting

- **Camera not working?** Check browser permissions
- **Slow performance?** Lower quality settings
- **Extension not loading?** Reload in chrome://extensions/

See [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) for more help.

## Tips

1. Use clear, well-lit photos for best results
2. Front-facing images work best
3. For low-end devices, use LITE version
4. Lower resolution improves performance
